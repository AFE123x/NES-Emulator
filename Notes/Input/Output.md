# I/O

- I/O is how your computer commuinicates with the outside world. 


## Memory mapped I/O

- memory-mapped I/O uses the address space to address main memory and I/O devices.
    - The memory and registers of I/O devices are mapped to address values.
        - AKA, a memory address can refer to physical ram or to the register of an I/O device.

- The range of addresses can be determined by hardware.
